Enunciado

Marcos quer comparar a soma das falhas em máquinas ímpares com a soma das falhas em máquinas pares para ver qual grupo apresenta mais falhas. O índice é o que determina se a máquina é par ou ímpar. Escreva uma função chamada comparar_somas que receba uma lista de números inteiros e retorne uma string indicando qual grupo (ímpares ou pares) tem a soma maior. Se as somas forem iguais, retorne "Iguais".

Entrada: Uma lista de números inteiros.

Saída: Uma string: "Ímpares", "Pares" ou "Iguais".
